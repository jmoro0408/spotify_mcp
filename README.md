# spotify_mcp



## Currently supported actions

1. Play a song
2. Get users playlists
3. Play from user playlist
4. Play from SP playlists like charts
5. Pause Playback
6. Skip to next
7. Skip to previous
8. Turn shuffle on
9. Turn shuffle off
10. Play a user's liked songs
11. Get recently played song
12. Add to queue
